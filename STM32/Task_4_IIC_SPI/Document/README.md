  - 了解IIC总线
       - SDA串行时钟线，用于同步通信。
       - CLK串行数据线，用于传输数据。
  - 了解SPI总线  
       - CLK提供同步时钟信号。
       - MOSI主设备向从设备发送数据。
       - MISO从设备向主设备发送数据。
       - DC区分数据和命令信号。
       - RW控制读写操作。
       - RST复位设备。
