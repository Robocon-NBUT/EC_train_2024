## &emsp;&emsp; <font face="华文黑体" color=#7D00FF>🍤认识i2c和spi

&emsp;&emsp;I2C ：一种多主机通信总线，用于连接微控制器和其他设备，如传感器和显示器。
SPI ：一种高速的，全双工，同步的通信总线，用于短距离通信。
